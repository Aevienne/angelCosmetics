# AngelCosmetics

Paper plugin project for Minecraft 1.21.x with:
- GUI cosmetics menu
- Trails
- Particle wings
- Recall animation
- Sample teleport command with recall effect

## Commands
- /cosmetics
- /recall
- /cosmeticsteleport <x> <y> <z>

## Build
Run:

```bash
./gradlew build
```

Then use the generated jar in `build/libs`.
